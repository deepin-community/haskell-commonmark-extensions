# Hard line breaks

This extension causes all newlines to be parsed as hard
line breaks rather than soft breaks.

```````````````````````````````` example
Hello
there
.
<p>Hello<br />
there</p>
````````````````````````````````

```````````````````````````````` example
*Hello 
there*
.
<p><em>Hello<br />
there</em></p>
````````````````````````````````
